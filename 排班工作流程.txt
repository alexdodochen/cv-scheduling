============================================
  心臟內科值班排班工作流程
============================================

【準備】
  - 排班.xlsx（含各月班表、值班總數統計、當月預班）
  - 確認當月國定假日
  - 確認展瀚醫師當月可值班數

--------------------------------------------
步驟 1：開始排班
--------------------------------------------
  對 Claude 說：
    「排 202605 的班」或「開始排五月的班」

  Claude 會自動：
    1. 計算當月平日班、假日班總數
    2. 讀取「值班總數統計」工作表（累計數據）
    3. 讀取「當月預班」工作表（偏好/避開日期）
    4. 詢問「展瀚醫師本月預計值幾個平日班？」

--------------------------------------------
步驟 2：回報展瀚班數
--------------------------------------------
  對 Claude 說：
    「展瀚值 3 班」

  Claude 會自動計算並回報：
    - 班數分配總覽（CR / 展瀚 / 建寬 / 每位 VS）
    - 各 VS 依累計假日班數排序，少的多值一班

  分配優先順序：
    1. CR（麒翔、見賢、常胤）：固定 15 平日 + 6 假日
    2. 展瀚：依回報的平日班數
    3. 建寬：吃剩餘平日班，最多 3 班，不值假日
    4. VS（廖瑀、昭佑、朝允、則瑋）：分剩餘的平日 + 假日班

--------------------------------------------
步驟 3：確認偏好與資訊
--------------------------------------------
  Claude 會呈現：
    - 各醫師累計班數是否正確
    - 各醫師首選/避開日期是否正確
    - VS 班數分配是否合理

  使用者確認後 Claude 才會開始排班演算。

--------------------------------------------
步驟 4：產生班表
--------------------------------------------
  Claude 會自動：
    1. 用 backtracking 演算法排班
    2. 在 排班.xlsx 建立當月工作表（日曆格式，假日底色標記）
    3. 產生「2026XX 班數統計」工作表
    4. 更新「值班總數統計」工作表（累加數據）

--------------------------------------------
排班規則摘要
--------------------------------------------
  - 假日班：週六、週日、國定假日
  - 平日班：非假日的週一至週五
  - 除展瀚外，所有人不可連續兩天值班
  - 盡量避免隔日值班（如週三值班就避免週一或週五）
  - VS 首選日期 > 展瀚/建寬首選 > CR 避開日期

  統計分類（用於平衡）：
  - 週五班：非假日週五，或連假前一天
  - 週六班：週六，或連假中間日
  - 週日班：週日，或連假最後一天

--------------------------------------------
建立新月份腳本
--------------------------------------------
  對 Claude 說：
    「建立六月排班腳本」

  Claude 會：
    1. 複製上月腳本，更新月份、假日、連假邏輯
    2. 從排班.xlsx 讀取最新累計統計作為 baseline
    3. 更新 fixed / avoid 偏好

--------------------------------------------
快速指令
--------------------------------------------
  「排 202605 的班」      → 開始五月排班流程
  「展瀚值 3 班」         → 回報展瀚班數，自動算 VS
  「開始排班」            → 確認後執行演算
  「更新排班統計」        → 重新計算並寫入統計表

--------------------------------------------
醫師名單
--------------------------------------------
  CR（總醫師）：麒翔、見賢、常胤
  VS（主治）  ：廖瑀、昭佑、朝允、則瑋
  中級        ：展瀚（可連值）、建寬（最多 3 平日/月）
