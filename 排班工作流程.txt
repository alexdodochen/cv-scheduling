============================================
  心臟內科值班排班工作流程
============================================

【準備】
  - Google Sheet「排班 GoogleSheet」（含各月班表、值班總數統計、當月預班）
    ID: 10ilVOmJrr8jjfnMMbtj60tAIIAe1YX3ZRU1RLgn6Elk
  - 本機需有 .gsa.json 服務帳號憑證（gitignored）
  - 確認當月國定假日
  - 確認展瀚醫師當月可值班數

--------------------------------------------
步驟 1：開始排班
--------------------------------------------
  對 Claude 說：
    「排 202605 的班」或「開始排五月的班」

  Claude 會自動：
    1. 計算當月平日班、假日班總數
    2. 讀取「值班總數統計」工作表（累計數據）
    3. 讀取「當月預班」工作表（偏好/避開日期）
    4. 詢問「展瀚醫師本月預計值幾個平日班？」

--------------------------------------------
步驟 2：回報展瀚班數
--------------------------------------------
  對 Claude 說：
    「展瀚值 3 班」

  Claude 會自動計算並回報：
    - 班數分配總覽（CR / 展瀚 / 建寬 / 每位 VS）
    - 各 VS 依累計假日班數排序，少的多值一班

  分配優先順序：
    1. CR（麒翔、見賢、常胤）：固定 15 平日 + 6 假日
    2. 展瀚：依回報的平日班數
    3. 建寬：吃剩餘平日班，最多 3 班，不值假日
    4. VS（廖瑀、昭佑、朝允、則瑋）：分剩餘的平日 + 假日班

--------------------------------------------
步驟 3：確認偏好與資訊
--------------------------------------------
  Claude 會呈現：
    - 各醫師累計班數是否正確
    - 各醫師首選/避開日期是否正確
    - VS 班數分配是否合理

  使用者確認後 Claude 才會開始排班演算。

--------------------------------------------
步驟 4：產生班表
--------------------------------------------
  Claude 會自動：
    1. 用 backtracking 演算法排班
    2. 在 Google Sheet 建立當月分頁（日曆格式，假日底色標記）
    3. 產生「2026XX 班數統計」分頁
    4. 更新「值班總數統計」分頁（累加數據）

--------------------------------------------
排班規則摘要
--------------------------------------------
  班別定義（以「值班總數統計」欄位為準）：
  - 平日班（一至四）：週一至週四，且非國定假日
  - 週五班：非國定假日的週五（或連假前一天）
  - 週六班：週六（或連假中間日）
  - 週日班：週日（或連假最後一天）
  - 假日班：週六 + 週日 + 週一至週五的國定假日
  注意：平日班「不含」週五，週五班獨立計算

  固定班數（不可更動）：
  - CR（麒翔、見賢、常胤）：合計 15 平日（含週五） + 6 假日（每人 5+2）
  - 建寬：最多 3 平日/月（實際上限依剩餘平日班數調整）

  排班約束：
  - 除展瀚外，所有人不可連續兩天值班（No Back-to-Back）
  - 盡量避免隔日值班 QOD（如週三值班就避免週一或週五）
  - VS 首選日期 > 展瀚/建寬首選 > CR 避開日期

  平衡原則：
  - CR 與 VS 皆依「值班總數統計」累計班數排序
  - 累計較少者優先排入，班數無法整除時少的多值一班

  統計分類（用於平衡追蹤）：
  - 週五班：非假日週五，或連假前一天
  - 週六班：週六，或連假中間日
  - 週日班：週日，或連假最後一天

--------------------------------------------
建立新月份腳本
--------------------------------------------
  對 Claude 說：
    「建立六月排班腳本」

  Claude 會：
    1. 複製 generate_schedule_202605.py（動態 baseline 版）
       — 不要複製 202604，那是舊版寫死 baseline
    2. 更新月份、假日、連假邏輯（get_stat_type）
    3. 更新 fixed / avoid 偏好
    baseline 會在執行時自動從「值班總數統計」載入，
    不用手動填 dict。

  重要：同一個月不可重跑腳本
  腳本會把本月加到「值班總數統計」裡。若重跑，會把本月
  算進自己的 baseline 而重複累計。要重跑請先把該月從
  累計表中扣掉，或從歷史基準 + 所有月度統計重新推算。

--------------------------------------------
快速指令
--------------------------------------------
  「排 202605 的班」      → 開始五月排班流程
  「展瀚值 3 班」         → 回報展瀚班數，自動算 VS
  「開始排班」            → 確認後執行演算
  「更新排班統計」        → 重新計算並寫入統計表

--------------------------------------------
Google Sheet 分頁命名規則
--------------------------------------------
  YYYYMM              → 日曆格式班表（如 202604）
  當月預班             → 醫師偏好/避開日期（每月覆寫）
  YYYYMM 班數統計     → 當月統計明細（如 202604 班數統計）
  值班總數統計         → 全期間累計班數（腳本動態讀寫）
  202508-202603 統計  → 歷史基準（不可變），從 CV班表.xlsx 重算而來

--------------------------------------------
醫師名單
--------------------------------------------
  CR（總醫師）：麒翔、見賢、常胤
  VS（主治）  ：廖瑀、昭佑、朝允、則瑋
  中級        ：展瀚（可連值）、建寬（最多 3 平日/月）

  ※ 姓名注意：CV班表.xlsx 歷史資料中「昭佑」曾被寫為
    「昭祐」（祐 vs 佑），腳本統計時已自動合併為同一人。
